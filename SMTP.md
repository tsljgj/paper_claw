Do not store live SMTP credentials in this repository.

Use GitHub Secrets instead:

- `SMTP_HOST`
- `SMTP_PORT`
- `SMTP_USER`
- `SMTP_PASS`
- `EMAIL_TO`

Example only:

- `SMTP_HOST=smtp.example.com`
- `SMTP_PORT=465`
- `SMTP_USER=bot@example.com`
- `SMTP_PASS=<app-password>`
- `EMAIL_TO=team@example.com`
